# ⚛️💨 Interactive Physics Learning Portal
### รายวิชา ฟิสิกส์ 5 (ว33205) | ฟิสิกส์ ม.6 บทที่ 16.3 ทฤษฎีจลน์ของแก๊ส (Kinetic Theory of Gases)

โปรเจกต์สื่อการสอนแบบโต้ตอบ (Interactive Web Application) ที่ถูกออกแบบมาเพื่อช่วยให้นักเรียนระดับชั้นมัธยมศึกษาปีที่ 6 เข้าใจความสัมพันธ์ระดับจุลภาคและมหภาคของแก๊สอุดมคติ ผ่านการจำลองภาพ (Simulation) การฝึกทำโจทย์แบบสุ่มตัวเลข และระบบสอบเก็บคะแนนแบบจับเวลา ตัวแอปพลิเคชันถูกรวมไว้ในไฟล์ HTML เพียงไฟล์เดียว (Single-File HTML) เพื่อความสะดวกในการใช้งานแบบออฟไลน์หรือนำไปโฮสต์บนเว็บเซิร์ฟเวอร์ทั่วไป

---

## ✨ ฟีเจอร์หลัก (Key Features)

### 1. 📖 ระบบทบทวนบทเรียนแบบโต้ตอบ (Interactive Review)
* **Simulator 1: แหล่งกำเนิดความดันแก๊ส** - แบบจำลอง Canvas 2D ที่แสดงให้เห็นการเคลื่อนที่ชนผนังของโมเลกุลแก๊สซึ่งก่อให้เกิดความดัน ($P$) ผู้เรียนสามารถปรับจำนวนโมเลกุล ($N$) และอัตราเร็ว เพื่อดูผลกระทบต่ออัตราการชนและความดันได้แบบ Real-time
* **Simulator 2: พลังงานจลน์เฉลี่ย** - อธิบายความสัมพันธ์ที่อุณหภูมิแปรผันตรงกับพลังงานจลน์เฉลี่ย ($\bar{E}_k$) โดยไม่ขึ้นกับชนิดของแก๊ส
* **Simulator 3: เปรียบเทียบอัตราเร็ว RMS** - แอนิเมชันเปรียบเทียบความเร็วของแก๊สต่างชนิดที่อุณหภูมิเดียวกัน (แก๊สเบาเคลื่อนที่เร็วกว่าแก๊สหนัก)
* **เครื่องคิดเลขคำนวณ $v_{rms}$** - เครื่องมือคำนวณที่แสดงการแทนค่าตัวเลขลงในสมการ (Substitution) ด้วยฟอร์แมตคณิตศาสตร์ที่สวยงาม

### 2. 🧠 ระบบตะลุยโจทย์ (Dynamic Practice Mode)
* **ระบบสุ่มตัวแปรในโจทย์ (RNG)** - ทำให้ผู้เรียนได้ฝึกคิดเลขและคำนวณโจทย์ที่ไม่ซ้ำกัน
* **คลังโจทย์และการค้นหา/กรอง (New!)** - สามารถค้นหาและกรองโจทย์แยกตามหัวข้อย่อย 16.3.1 (ความดัน), 16.3.2 (พลังงานจลน์เฉลี่ย), 16.3.3 (อัตราเร็ว RMS) และประเภทของโจทย์ (ปรนัย/คำนวณ) เพื่อเจาะลึกเฉพาะหัวข้อได้
* **ระบบตรวจคำตอบทันที** - พร้อมเฉลยวิธีทำทีละขั้นตอน (Step-by-step Explanation) ที่ใช้สมการ LaTeX ที่อ่านง่าย

### 3. ⏱️ ระบบสอบเก็บคะแนน (Timed Exam System)

* **ระบบจำลองการสอบ** - จำนวน 5 ข้อ (10 คะแนน) พร้อมตัวจับเวลาถอยหลัง (Timer)
* **ระบบป้องกันการโกง (Anti-cheat guard)** - ป้องกันการออกจากหน้าสอบหรือรีเฟรชหน้าจอ
* **Dashboard สรุปผล** - สรุปคะแนน ความถูกต้องของแต่ละข้อ และแสดงวิธีทำอย่างละเอียด
* **ระบบบันทึกผลคะแนน** - บันทึกผลสอบล่าสุดไว้ในเบราว์เซอร์ (Local Storage) เพื่อให้ผู้เรียนกลับมาดูความคืบหน้าได้

### ## 🎯 Dynamic Quiz System Rules (กฎระบบข้อสอบแปรผันตามเลขที่)

ระบบข้อสอบในโปรเจกต์นี้ทั้งหมด (ทุกการสอบย่อยและสอบเก็บคะแนน) ออกแบบให้เป็น **Dynamic System** โดยผูกค่าตัวแปรในโจทย์เข้ากับ **"เลขที่ของนักเรียน (Roll Number: $R$)"** เพื่อให้มั่นใจว่านักเรียนเลขที่ 1-40 จะได้ค่าพารามิเตอร์และคำตอบสุดท้ายที่ไม่ซ้ำกันเด็ดขาด (Unique Answers)

### 📌 Core Architecture & Logic for AI Agent:
1. **Dynamic Parameter:** หน้าบ้าน (Frontend) จะส่งค่า `R` (Roll Number) มาเพื่อเป็น seed ในการคำนวณสร้างโจทย์
2. **Backend Validation:** การตรวจสอบความถูกต้องของคำตอบต้องคำนวณ On-the-fly จากเลขที่ `R` ห้ามใช้ค่าคำตอบคงที่ (Static Hardcoded)
3. **Safety Constraints:** การสุ่มพารามิเตอร์ที่ผูกกับ $R$ ต้องไม่ทำให้คำตอบติดลบ หรือเกิดค่าที่ไม่สมเหตุสมผลทางฟิสิกส์

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

* **HTML5 / CSS3 / JavaScript (ES6+)**: โครงสร้างหลักและลอจิกทั้งหมดถูกเขียนด้วย Vanilla JS โดยไม่พึ่งพา Framework หนักๆ ทำให้ทำงานได้เร็วและเบามาก
* **Tailwind CSS (via CDN)**: ใช้ในการจัดการ UI/UX, สีสัน, Layout และความ Responsive ให้สวยงามทันสมัย รองรับทั้งหน้าจอคอมพิวเตอร์และมือถือ
* **MathJax**: ไลบรารีสำหรับเรนเดอร์สมการคณิตศาสตร์และสูตรฟิสิกส์ที่เขียนด้วยไวยากรณ์ LaTeX ให้แสดงผลบนเว็บได้อย่างสมบูรณ์แบบ
* **FontAwesome**: ใช้สำหรับไอคอนต่างๆ ในแอปพลิเคชัน
* **Google Fonts**: ใช้ฟอนต์ 'Prompt' และ 'Sarabun' เพื่อให้อ่านง่ายและมีความเป็นวิชาการ

## 🚀 วิธีการใช้งาน (How to Run)

เนื่องจากโปรเจกต์นี้เป็น **Single-File HTML** คุณจึงไม่จำเป็นต้องติดตั้ง Node.js, ฐานข้อมูล หรือ Server ใดๆ
1. ดาวน์โหลดไฟล์ `index.html` (หรือเปิดโดยตรง)
2. ดับเบิลคลิกเปิดไฟล์ด้วยเว็บเบราว์เซอร์ใดก็ได้ (เช่น Google Chrome, Microsoft Edge, Safari)
3. ระบบพร้อมใช้งานทันที! *(ต้องใช้การเชื่อมต่ออินเทอร์เน็ตในการโหลดฟอนต์, Tailwind CSS และ MathJax ในครั้งแรก)*
